Sci-ModoM


is a unique resource to elucidate on a transcriptome-wide scale the impact and the role of processing and modification on RNA function.

Sci-ModoM v0.0.0-alpha development server has limited functionality.

It is possible to perform queries to determine if a site is subject to modification or processing events for a test dataset.

Sci-ModoM

, a project of RMap (RNA Modification and Processing), aims to:

  • constitute a reference database for high-throughput transcriptome-wide RNA modification mapping data
  • integrate RNA processing events and other relevant public data
  • harmonize transcriptome-wide data exchange formats and interoperability standards
  • provide a critical assessment of and integration with public data
